## Overview

Supervised learning is a machine learning approach where models are trained on **labeled data** to learn patterns and make predictions on unseen data. ([Wikipedia][1])

This repository serves as a collection of:

* Practice implementations
* Lab experiments
* Model training workflows
* Performance evaluation
